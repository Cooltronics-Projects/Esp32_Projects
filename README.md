# ⚡ ESP32 Projects

A collection of DIY projects built around the **ESP32 microcontroller** — covering IoT, home automation, AI integration, wireless communication, and more. Each project lives in its own folder with full source code, wiring diagrams, and documentation.

---

## 📺 About This Repo

This repository is the companion to the **[Cooltronics Projects](https://www.youtube.com/@CooltronicsProjects)** YouTube channel — where we build real, working ESP32 projects using affordable components available in India.

Every project here is:
- ✅ Beginner-friendly
- ✅ Built and tested on real hardware
- ✅ Fully documented with code + wiring

## 🛠️ Common Components Used

- ESP32 Dev Board
- ESP32-CAM
- OLED Displays (SSD1306, SSD1309)
- DHT11 / DHT22 Sensors
- MPU6050 Gyroscope
- Relay Modules
- Servo & Stepper Motors
- MQTT / HTTP / WebSocket Communication

## 🚀 Getting Started

1. Clone this repo:
   ```bash
   git clone https://github.com/Cooltronics-Projects/Esp32_Projects.git
   ```
2. Navigate into any project folder
3. Open the project's `README.md` for wiring + setup instructions
4. Upload the code using Arduino IDE or PlatformIO

---

## 📌 Requirements

- [Arduino IDE](https://www.arduino.cc/en/software) or [PlatformIO](https://platformio.org/)
- ESP32 board package installed in Arduino IDE
  - Board URL: `https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json`
- Required libraries listed in each project's `README.md`
